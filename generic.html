IST707
Analytics & Machine Learning

The data set selected for analysis in this project is related to prediction of red wine quality. The dataset was obtained from Kaggle. (Red Wine Quality, 2021). It is a beginner level dataset that contains 11 attributes and 1599 clean records related to the quality and variants of the Portuguese “Vinho Verde” red wine. The final analysis submission consists of this word report, csv dataset and a rmd file containing the R code used for the analysis. This document contains an Exploratory Data Analysis (EDA), model selection and model runs with an accuracy metric to determine the success of the models.

  • IST707 Project

IST719
Information Visualization

The dataset for this analysis consists of six csv files that are combined on common attributes. The two files contain information on Movies and TV shows from Kaggle (Basnal, 2021) on the streaming platforms: Netflix, Hulu, Disney and Amazon Prime. The additional four csv files contained genre information by streaming service platform titles from Kaggle (Joshi, 2021). Content was evaluated on age, genre, vintage, content type and rotten tomatoes rating. The munging, EDA, analysis and visualizations were created in R. The final poster was compled and edited in Adobe Illustrator.

  • IST719 Project

IST718
Big Data Analytics

The final group project for this course my team identified a data-focused problem, centering on health care and the risk factors and demographics for diagnosis of heart disease. For this project I preformed an association rules mining excersise and created a Nural Network model that predicted with 100% accuracy and 0% error for the dataset.
